# React Router App

A React app using `react-router-dom` for client-side navigation.

## Structure
- `src/pages/Home.jsx`: The default landing page.
- `src/pages/About.jsx`: The about page.
- `src/pages/Section.jsx`: A dynamic page showing route parameters.
- `src/components/Navbar.jsx`: Links to navigate between pages.
- `src/App.jsx`: Contains the routing logic.
