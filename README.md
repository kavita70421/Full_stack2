# Experiments & Learning Projects

A collection of learning experiments covering full-stack web development, including React, Node.js, Java, and various JavaScript frameworks.

## 📁 Project Structure

### Experiments (exp - 1 to exp - 10)

| Experiment | Description | Tech Stack |
|-----------|-------------|-----------|
| **exp - 1** | Basic React setup with Navbar and Contact components | Vite, React, JavaScript |
| **exp - 2** | React with Dashboard and Form components | Vite, React, JavaScript |
| **exp - 3** | Routing with About, Home, and Section pages | Vite, React, React Router |
| **exp - 4** | State management with Redux and Theme Context | Vite, React, Redux Toolkit, Context API |
| **exp - 5** | Advanced React components and features | Vite, React, JavaScript |
| **exp - 6** | Create React App setup with testing | Create React App, React, JavaScript |
| **exp - 7** | Spring Boot Java backend | Java, Maven, Spring Boot |
| **exp - 8** | MERN Stack with frontend and backend | React, Express, Node.js, MongoDB |
| **exp - 9** | React application with CSS and components | Create React App, React, JavaScript |
| **exp - 10** | Node.js backend with Student routes and models | Express, Node.js, MongoDB |

## 🛠️ Tech Stack Overview

- **Frontend**: React, Vite, Create React App
- **Backend**: Node.js, Express.js, Spring Boot
- **Databases**: MongoDB
- **State Management**: Redux, Context API
- **Build Tools**: Webpack, Vite, Maven
- **Testing**: Jest, React Testing Library
